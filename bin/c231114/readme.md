### 21일 과제 : 예외처리

- 조건

  - ID
    - 길이 : 3 ~ 20
    - 영어 or 숫자
  - PW
    - 길이 : 10~30
    - 대소문자, 숫자, 특수문자(!@#$%^&) 포함
  - 이름
    - 한글 길이 : 2~10
    - 영어 길이 : 4~20
  - 전화번호
    - "-"를 포함한 12~13 
    - "-"를 제외한 10~11
    - DB에는 "-" 포함하여 저장
  - 주소(입력 시)
    - 길이 : 5~100
  - Email
    - .com || .org || .co.kr || .net 으로 종료될 것
    - 중간에 @가 들어갈 것
  - 깃주소(입력 시)
    - 아이디만 입력 시 "https://github.com/" 포함하여 저장
